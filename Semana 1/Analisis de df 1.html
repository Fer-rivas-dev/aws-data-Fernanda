Este es el analisis de Spotify Analysis Dataset 2025

Esta gua te explica qu hacer y por qu en cada paso, lista de comandos y fragmentos de cdigo Python (pensado para ejecutar en Jupyter Notebook o en un script). Est orientada al dataset con columnas:

['user_id', 'gender', 'age', 'country', 'subscription_type', 'listening_time', 'songs_played_per_day', 'skip_rate', 'device_type', 'ads_listened_per_week', 'offline_listening', 'is_churned']

  1. Requisitos / Preparacin

  1. Tener un entorno virtual activo (opcional pero recomendado): python3 -m venv venv source venv/bin/activate

  2. Instalar libreras necesarias (ejecutar en terminal o celda de Jupyter con !): pip install pandas numpy matplotlib seaborn

3.Descargar el CSV y colocarlo en la carpeta de trabajo (p. ej. spotify_dataset.csv) en https://www.kaggle.com/datasets/nabihazahid/spotify-dataset-for-churn-analysis.
